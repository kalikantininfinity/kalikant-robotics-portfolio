# Kalikant Jha Robotics Portfolio

A complete static portfolio website for robotics, computer science, embedded AI, and the AMDVS project.

## Files

- `index.html` — website structure and content
- `styles.css` — full visual design, responsiveness, animations
- `script.js` — live rover simulation, HUD animation, mobile navigation
- `assets/` — favicon, social preview, and custom SVG illustrations
- `netlify.toml` — Netlify publish and basic headers
- `_redirects` — static fallback routing
- `robots.txt` and `sitemap.xml` — basic SEO files

## Local preview

Run one of these commands in this folder:

```bash
python -m http.server 5500
```

Then open:

```text
http://localhost:5500
```
