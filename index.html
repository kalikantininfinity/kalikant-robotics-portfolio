Roadmap

Three-phase build plan

The current direction reflects the upgraded phase order: ground autonomy first, LLM integration second, drone cooperation third.

01

Autonomous ground rover

ROS 2, OpenCV, YOLOv8, TensorRT, camera perception, lane following, Nav2, sensor fusion, and safety watchdogs.

  • Real-time object detection
  • Waypoint navigation
  • LiDAR / GNSS / IMU roadmap
02

LLM decision support

High-level reasoning layer that reads system state, summarizes events, explains failures, and helps choose safe next actions.

  • Mission memory and logs
  • RAG-style project documentation
  • Human-readable decision reports
03

Cooperating drone module

Later expansion: overhead reconnaissance, AprilTag/ArUco docking, and return-to-rover coordination.

  • Drone launch and return workflow
  • Marker-based precision landing
  • Ground-air coordination

Builder philosophy

Proof beats polish. Logs beat guesses.

Builder before anything else. I learn by making, breaking, measuring, and rebuilding.

Safety is part of intelligence. A robot is not smart if it cannot fail safely.

Every project is a brick. Reusable code, documented experiments, and systems I can explain end to end.

Self-funded means disciplined. No waste, no fake complexity, no black boxes I cannot debug.
